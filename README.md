# KINEMA

**Kalman Integrated Navigation for Experimental Motion Analysis**

An interactive physics learning platform for exploring kinematics. Built with React, Vite, TypeScript, and Tailwind CSS.

## Features

- Kinematic calculator (horizontal, vertical, projectile)
- Real-time motion animations with selectable objects and backgrounds
- Motion graphs explorer with vector addition playground
- Step-by-step derivations of the four kinematic equations
- Camera-based object motion tracker with gate-line timing

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:8080.
