# Derivation Tab Font Size Reduction

Reduce font sizes across the Derivation tab so content is comfortable and does not overwhelm readers.

## Changes

- **Main title**: `text-4xl md:text-5xl` → `text-3xl md:text-4xl`
- **Section headings**: `text-3xl md:text-4xl` → `text-2xl md:text-3xl`
- **Intro / guide paragraphs**: `text-lg md:text-xl` / `text-lg` → `text-base md:text-lg`
- **Step body text**: `text-lg leading-relaxed` → `text-base leading-relaxed`
- **Formula blocks (small)**: `text-xl md:text-2xl` → `text-lg md:text-xl`
- **Formula blocks (big)**: `text-3xl md:text-4xl` → `text-2xl md:text-3xl`
- **Example / callout text**: `text-base` stays `text-base`
- **All four together grid**: `text-xl` → `text-lg`
- **"All four together" heading**: `text-2xl` → `text-xl`
- **Summary paragraph**: `text-base` stays `text-base`

No other files touched.
